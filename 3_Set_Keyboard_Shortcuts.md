## Setting some useful Keyboard Shortcuts

1. Go to Settings -> Keyboard Shortcuts -> Click on + 

### To Set Terminal Shortcut :-
    Name : Terminal (Any Name)
    command : gnome-terminal
    shortcut : CTRL + ALT + T
    
### To Set Files Shortcut (Like Windows) :-
    Name : Files (Any Name)
    command : nautilus
    shortcut : SUPER + E
